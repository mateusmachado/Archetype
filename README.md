## Front-End - Requirements to run

   Install nodeJS

# Start application:
  ###  Navigate to the directory path: $PATH/ui
  ## npm install less less-loader fish-ui --save

   Install Gradle
   Postgres 
     BaseName: fish
     
# Start application:

   run command line: gradle bootRun
